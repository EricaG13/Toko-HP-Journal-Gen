Class: Tier

TokoHPApp.Tier(name, hp_max, overflow_from_prev)

new Tier(name, hp_max, overflow_from_prev)

A specialized list of Artworks. Each Tier has a maximum HP which its component Artworks can add up to - once this value is passed, no more Artworks can be added to the Tier. The amount by which a Tier surpasses its HP maximum is considered the Tier's overflow. A Tier should be initialized with the overflow from the previous Tier.
Parameters:
Name Type Description
name string The display name of this Tier.
hp_max number The maximum HP value this Tier can hold.
overflow_from_prev number Optional. The amount of HP which overflowed from the previous Tier into this one.
